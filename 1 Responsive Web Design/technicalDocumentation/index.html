HTML
  • HTML Elements

    HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.

  • HTML Attributes

    HTML attributes provide additional information about HTML elements. Attributes take the form of an opening tag and additional info is placed inside.

  • HTML Headings

    HTML headings are titles or subtitles that you want to display on a webpage. There are six levels of headings, from h1 to h6.

  • HTML Paragraphs

    HTML paragraphs are defined with the p tag. They are used to define a block of text that should be written as a paragraph.

  • HTML Links

    HTML links are defined with the a tag. The link's destination is specified in the href attribute.

  • HTML Images

    HTML images are defined with the img tag. The source file is specified in the src attribute.

  • HTML Lists

    HTML lists are defined with the ul, ol, and li tags. The ul tag creates an unordered list, the ol tag creates an ordered list, and the li tag creates a list item.

HTML Code Samples

Here are some code samples for HTML elements: 

     
      <h1>This is a heading</h1>
      <p>This is a paragraph</p>
      <a href="https://www.example.com">This is a link</a>
CSS
  • CSS Selectors

    CSS selectors are used to "find" (or select) the HTML elements you want to style. We can divide CSS selectors into five categories: Simple selectors, Combinator selectors, Pseudo-class selectors, Pseudo-element selectors, and Attribute selectors.

  • CSS Properties

    CSS properties are used to specify the style of an element, such as color, font, size, and more. A property is a pair of a key and a value.

  • CSS Values

    CSS values are set against properties and are either keywords or numerical values. There are different types of values, such as color values, length values, percentage values, etc.

  • CSS Syntax

    A CSS rule-set consists of a selector and a declaration block. The declaration block contains one or more declarations separated by semicolons.

CSS Code Samples

Here are some code samples for CSS styling: 

      
       h1 {
        color: blue;
        font-size: 26px;
       }
       p {
        color: green;
       }
JavaScript
  • JavaScript Variables

    JavaScript variables are containers for storing data values. You can think of them as symbolic names for values.

  • JavaScript Operators

    JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more.

  • JavaScript Functions

    JavaScript functions are blocks of code that can be defined once and executed whenever needed.

  • JavaScript Objects

    JavaScript objects are containers for named values called properties or methods.

JavaScript Code Samples

Here are some code samples for JavaScript functions: 

      
       function greet(name) {
        return "Hello, " + name + "!";
       }
       console.log(greet("World"));
JSON
  • JSON Syntax

    JSON syntax is derived from JavaScript object notation syntax, but it is entirely text-based. JSON exists as a string, and it needs to be converted to a JavaScript object when it is received.

  • JSON Data Types

    JSON data types are used to define the type of data that can be stored in a JSON object. There are six data types in JSON: string, number, object, array, boolean, and null.

  • JSON Objects

    JSON objects are written in key/value pairs. Keys must be strings, and values must be a valid JSON data type (string, number, object, array, boolean, or null).

  • JSON Arrays

    JSON arrays are written in square brackets and can contain multiple objects separated by commas.

JSON Code Samples

Here are some code samples for JSON objects: 

      
       {
        "name": "John",
        "age": 30,
        "city": "New York"
       }
